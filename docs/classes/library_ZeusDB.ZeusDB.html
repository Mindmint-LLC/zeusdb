    Staticconnect

    • connect(identifier: DataSourceIdentifier): Promise<ZeusConnection>

      Start a new connection to the data source. If the data source uses pooling, a pooled connection will be returned if one is available. Be sure to always call connection.close() when finished with the connection to release it back to the pool.

      Only one transaction can be active per connection at a time.

      You may also use the ZeusDataSource.connect() method directly to start a connection.

      Parameters

      • identifier: DataSourceIdentifier

        string or number matching the "dataSource.id" property.

      Returns Promise<ZeusConnection>

    StaticregisterDataSource

    • registerDataSource(config: IDbConfig): ZeusDataSource

      Register a new data source. Required to be called before using the data source. The data source is identified by the "id" property in the config, if the id is not provided, a UUID will be generated.

      Each data source can have its own configuration, including database provider, connection details, and pooling options.

      Parameters

      • config: IDbConfig
